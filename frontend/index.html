How to use

Enter realistic numeric values in the fields. The model expects numbers — strings will fail.
  • Budget: integer USD.
  • Runtime: minutes.
  • Popularity: dataset popularity score (float).
  • Vote Average: rating from 0 to 10.
  • Vote Count: number of ratings (integer).
